Korean Master

Description

Korean Master is an interactive multi-agent language tutor that guides you through Korean vocabulary, grammar drills and conversational practice. It adapts difficulty based on your performance and provides real-time corrections.

Model

Built on a fine-tuned GPT-3.5 architecture with a custom dialog tree. Supports both text and audio prompts, with on-the-fly speech-to-text integration.

Quick Start (README)

  • 1. Click “Start Lesson” to launch the first interactive module.
  • 2. Use commands like /translate 안녕하세요 for instant translations.
  • 3. Track progress in your dashboard and review flashcards.
